# Containerboard Throughput Optimization — Lean Six Sigma Green Belt Project

**Author:** Maximillian Ermler, Plant Optimization Engineer / Quality and Safety Manager  
**Certification:** Lean Six Sigma Green Belt (PMI)

## Overview

Full DMAIC Lean Six Sigma project targeting throughput improvement on the Goepfert 282 3-color Die Cutter at Packaging Corporation of America. Baseline MSF/Machine Hr averaged 21.73 against a target of 30.42 — a 40% improvement gap. The project delivered a 22% improvement and $863,700 in verified contribution.

---

## DMAIC Methodology

### Define
- Scoped throughput improvement on the 282 D/C containerboard order fulfillment process
- Established project charter with champion, team, baseline metrics, and financial targets
- Baseline: 21.73 MSF/Machine Hr · Target: 30.42 MSF/Machine Hr (+40%)

### Measure
- Audited measurement systems for MSF/Machine Hr using CTI/EFI data
- Validated calculation methodology against direct observation
- Excluded downtime and setup time data due to accuracy concerns — documented rationale
- Generated process control charts (MSF/Machine Hr vs. Week) to establish baseline variation

### Analyze
- Developed high-level and detailed process maps across full order fulfillment workflow
- Conducted waste walk focusing on setup operations — identified wasted crew motion
- Built spaghetti diagram to visualize movement inefficiencies during setup
- Performed root cause analysis via Cause & Effect (Ishikawa) diagram
- Prioritized root causes using Cause and Effects Matrix
- Built Pareto chart identifying top three sources of machine downtime
- Developed preliminary FMEA to assess failure modes and risk priority numbers

### Improve
- Developed and implemented new SOP for Goepfert 282 D/C setup procedure
- Modeled bundle breaker behavior to optimize machine sequencing
- Updated Corporate Knowledge Base with revised machine procedure
- Implemented crew training on new standardized setup process

### Control
- Conducted post-improvement normality test (Anderson-Darling) — confirmed normal distribution
- Calculated new process capability: Cp = 1.46, Cpk = 0.96
- Developed control plan for ongoing manufacturing monitoring
- Documented proof of sustainment with month-over-month tracking
