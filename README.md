# lean-six-sigma-throughput-optimization
Lean Six Sigma Green Belt project increasing throughput 22% on a Goepfert 282 die cutter at Packaging Corporation of America. Full DMAIC methodology: process mapping, MSA, FMEA, root cause analysis, capability analysis (Cp/Cpk), and control planning. Resulted in $863,700 contribution (more in sales volume). PMI certified.
